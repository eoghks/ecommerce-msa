## 미통과 항목 상세

### #10 — 다운스트림 연결 실패 시 500 반환

**원인**  
`GatewayExceptionHandler` 미존재. product-service 연결 거부(`ConnectException`) 발생 시 Spring Cloud Gateway 기본 동작이 500으로 응답.

**수정 내용**  
`fix/gateway-upstream-error-handling` 브랜치 생성 (PR #3)

```java
// GatewayExceptionHandler.java 신규 추가
@Order(-1)
@Component
public class GatewayExceptionHandler implements ErrorWebExceptionHandler {
    // ConnectException → 503 Service Unavailable
    // PrematureCloseException → 502 Bad Gateway
    // 기타 → 500
}
```

**재테스트 결과**

```
GET /api/v1/products (product-service 미기동, 유효 JWT)
→ HTTP 503
→ {"status":503,"error":"Service Unavailable","message":"서비스를 일시적으로 사용할 수 없습니다.","path":"/api/v1/products"}
```

✅ **수정 후 통과** (PR #3 develop 머지 완료)
